var

    • var 无块级作用域（块语句的作用域）：

    • var 存在函数作用域：

      在JavaScript中每个函数作为一个作用域，在外部无法访问内部作用域中的变量。

    • 函数出现嵌套，var 出现作用域嵌套链：

      可能会出现多条并行的作用域嵌套链，在使用变量的时候要去判断属于哪一条。

    • 作用域链执行前已创建：

      上述代码，在函数被执行之前已经创建了两条作用域链：

      全局作用域 -> Bar函数作用域

      全局作用域 -> Func函数作用域

      当执行【ret();】时，ret代指的Bar函数，所以根据“全局作用域 -> Bar函数作用域”去判定，这时候xo = ‘alex’。

    • 提前声明：

      JavaScript的函数在被执行之前，会将其中的变量全部声明，而不赋值。所以将声明放在赋值之后编译是可以通过的。

let

  • let声明一个变量，它使用的是词法作用域或块作用域

  • 暂时性死区：在 let变量声明之前，不能去对它进行操作，但是可以获取它（但是运行时会抛出错误）。

  • let变量可以在不同的块级定义域中重新定义，而且会屏蔽定义域外的定义。

const

  • const定义的变量不能被直接修改，但是里面的属性可以被修改，这点和java类似。
