chapter1-Introduction to database

1.The Characteristics of Databases

  • 数据库的目的：

    The purpose of a database is to help people track of things.

  • 最常用的数据库：

    The most commonly used type of database is the relational database.

  • 数据库包含数据和数据之间的关系

    A database contains both data and the relationships among the data.

  • 外键将多张表联系起来，外键都构成了数据库中的关系。

    A foreign key provides the link between two tables. By adding a foreign key, we can create a relationship between tables.

  • 表中的每一个记录通过主键来区别，所以主键用来在表之间建立关系。

    Each row in a table is uniquely indentified by a primary key, and the values of these keys are used to create the relationships between the tables.

  • 信息的定义：①从数据中推导出的知识。②出现在上下文中的数据。③进行过求和、排序、平均、分组、比较等操作的数据。

    Since we have already defined data as recorded facts and numbers, we can now define information as:

    • Knowledge derived from data.

    • Data presented in a meaningful context.

    • Data processed by summing, ordering, averaging, grouping, comparing or other similar operations.

2.Database Example

3.The Components of a Database System

  • 数据库系统的四大组件：用户，数据库应用，数据库管理系统，数据库

    As shown in Figure 1-6, a database system is typically defined to consist of four components: users, the database application, the database management system (DBMS), and the database.

  • 什么是数据库：数据库是有关联关系的表和其他结构的集合。

    The database is a collection of related tables and other structures.

  • 为什么数据库是自我描述的：因为数据库包含了它自己的描述。数据库不光有用户的数据表，还有描述用户数据的数据表。

    A database is self-describing because it contains a description of itself.

    Databases contain not only tables of user data, but also tables of data that describe that user data.

  • 什么叫做元数据：描述数据的数据叫做元数据。

    The descriptive data which is about data is called metadata. The form and format of metadata varies from DBMS to DBMS.

  • 什么是DBMS：DBMS是用来创建、运行、管理数据库的程序。DBMS接受用SQL编码的请求，然后将它们翻译成数据库的执行操作。DBMS通常是用软件提供商授权的一个大型、复杂的软件，大部分公司都不会自行去设计总结的DBMS。

    The database management system (DBMS) is a computer program used to create, process, and administer the database. The DBMS receives requests encoded in SQL and translates those requests into actions on the database. The DBMS is a large, complicated program that is licensed from a software vendor; companies almost never write their own DBMS programs.

  • DBMS的基本功能：①创建数据库。②创建表。③创建支持的结构（例如索引）。④读取数据库数据。⑤编辑（增，删，改）数据库数据。⑥执行规则。⑦并发性控制。⑧权限控制。⑨备份与恢复

    1. Create database
    2. Create tables
    3. Create supporting structures (e.g., indexes)
    4. Read database data
    5. Modify (insert, update, or delete) database data
    6. Maintain database structures
    7. Enforce rules
    8. Control concurrency
    9. Provide security
    10. Perform backup and recovery

  • 什么是数据库软件：数据库软件是一系列电脑软件，它们作为了用户和DBMS之间的中间者。数据库应用通过发送SQL命令读取或者编辑数据库中的数据。数据库应用也通过表单或者记录向用户展示数据。数据库应用即可从软件提供商中获取，也可以内部自行编写。

    A database application is a set of one or more computer programs that serves as an intermediary between the user and the DBMS. Application programs read or modify database data by sending SQL statements to the DBMS. Application programs also present data to users in the format of forms and reports. Application programs can be acquired from software vendors, and they are also frequently written in-house.

  • 数据库应用的基本功能：①创建和处理表单。②处理用户的查询。③创建和处理报告。④执行应用逻辑。⑤控制应用。

    1. Create and process forms
    2. Process user queries
    3. Create and process reports
    4. Execute application logic
    5. Control application

  • 什么是用户：用户通过一个数据库应用去追踪数据。他们通过表单来读取、插入或查询数据，他们也生产报告来传递数据。

    Users employ a database application to keep track of things. They use forms to read, enter, and query data, and they produce reports to convey information.

4.Personal Versus Enterprise-Class Database Systems

  • Microsoft Access 的误解：Microsoft Access 不仅仅是一个DBMS，它是一个个人的数据库系统：一个DBMS加一个应用产生器。

    Microsoft Access is not just a DBMS. Rather, it is a personal database system: a DBMS plus an application generator.

  • 企业级数据库系统中的数据库应用分类：①客户-服务器。②网页服务器上的电子贸易应用。③公司门户上的信息发布应用。④XML网页服务应用。

    1. Applications running over corperate network(Client-Server)
    2. E-Commerce applications on web server
    3. web portal with reporting applications
    4. XML web services applications

5.Database Design

Ⅰ.Database Design from Existing Data

  • 什么叫数据仓库：数据仓库保存了以研究或报告为目的的数据，这些数据是提供给其他分析工具的。

    The data warehouse and data mart databases store data specifically organized for research and reporting purposes, and these data often are exported to other analytical tools, such as SAS’s Enterprise Miner, IBM’s SPSS Data Modeler, or TIBCO’s Spotfire Metrics.

  • 相关联表的新建是分开为多张表还是合并为一张表：遵循规范化/范式的原则。

    Decisions such as this are not arbitrary. Database professionals use a set of principles, collectively called normalization, or normal forms, to guide and assess database designs.

Ⅱ.Database Design for New Systems Development

  • 从用户需求转换成数据库设计的步骤：首先根据用户需求设计出数据模型，然后将它们转换数据库模型。

    First, the team creates a data model from the requirements. statements and then transforms that data model into a database design.

Ⅲ.Database Redesign

  • 重新设计数据库的两种类型：①改编为新的/改变的数据库（数据库迁移）。②合并多个数据库
    1. a database is adapted to new or changing requirements. This
      process sometimes is called database migration.
    2. the integration of two or more databases.
