线程间同步

一. mutex互斥

Mutex变量是非0即1的，可看作一种资源的可用数量，初始化时Mutex是1，表示有一个可用资源，加锁时获得该资源，将Mutex减到0，表示不再有可用资源，解锁时释放该资源，将Mutex重新加到1，表示又有了一个可用资源。

mutex 用 pthread_mutex_t 类型的变量表示。

头函数:

#include<pthread.h>

初始化与销毁：

//返回值：成功返回0，失败返回错误号。
int pthread_mutex_init(pthread_mutex_t *restrict mutex,const pthread_mutexattr_t *restrict attr);
//参数attr设定Mutex的属性，如果attr为NULL则表示缺省属性。
pthread_mutex_t mutex = PTHREAD_MUTEX_INITIALIZER;
//如果Mutex变量是静态分配的（全局变量或static变量），也可以用宏定义PTHREAD_MUTEX_INITIALIZER来初始化：
int pthread_mutex_destroy(pthread_mutex_t *mutex);
//销毁

mutex的加锁解锁:

//返回值：成功返回0，失败返回错误号。
int pthread_mutex_lock(pthread_mutex_t *mutex);
//若mutex为1，则mutex减1加锁
//若mutex为0，说明另一线程已经调用pthread_mutex_lock获得了该Mutex，则当前线程挂起等待

int pthread_mutex_trylock(pthread_mutex_t *mutex
//若mutex为1，则mutex减1加锁
//若mutex为0，当前线程不会挂起等待，函数返回EBUSY

int pthread_mutex_unlock(pthread_mutex_t *mutex);
//mutex置1解锁

二、 Condition Variable 条件变量

Condition Variable用pthread_cond_t类型的变量表示。
一个Condition Variable总是和一个Mutex搭配使用的。

头函数：

#include<pthread.h>

初始化与销毁：

//返回值：成功返回0，失败返回错误号。
int pthread_cond_init(pthread_cond_t *restrict cond, const pthread_condattr_t *restrict attr);
//attr参数为NULL则表示缺省属性
pthread_cond_t cond = PTHREAD_COND_INITIALIZER;
//静态分配
int pthread_cond_destroy(pthread_cond_t *cond);
//销毁

使用：

int pthread_cond_wait(pthread_cond_t *restrict cond, pthread_mutex_t *restrict mutex);
//在调用pthread_cond_wait()前必须由本线程加锁（pthread_mutex_lock(),mutex保持锁定状态
//在条件不满足cond=0时mutex置1解锁，同时本线程挂起。
//当条件满足cond=1时mutex置0加锁，当前进程继续执行。

int pthread_cond_timedwait(pthread_cond_t *restrict cond, pthread_mutex_t *restrict mutex,   const struct timespec *restrict abstime);
//一个额外的参数可以设定等待超时，如果到达了abstime所指定的时刻仍然没有别的线程来唤醒当前线程，就返回ETIMEDOUT

int pthread_cond_broadcast(pthread_cond_t *cond);
//唤醒在这个Condition Variable上等待的所有线程。

int pthread_cond_signal(pthread_cond_t *cond);
//唤醒在某个Condition Variable上等待的另一个线程

三、Semaphore信号量

信号量（Semaphore）和Mutex类似，表示可用资源的数量，和Mutex不同的是这个数量可以大于1。
semaphore变量的类型为sem_t;

头函数：

#include <semaphore.h>

初始化与销毁：

int sem_init(sem_t *sem, int pshared, unsigned int value);
//pshared参数为0表示信号量用于同一进程的线程间同步,value参数表示可用资源的数量，
int sem_wait(sem_t *sem);
//获得资源，使semaphore的值减1,如果调用sem_wait()时semaphore的值已经是0，则挂起等待
int sem_trywait(sem_t *sem);
//果调用sem_wait()时semaphore的值已经是0，不会挂起等待。
int sem_post(sem_t * sem);
//释放资源，使semaphore的值加1，同时唤醒挂起等待的线程。
int sem_destroy(sem_t * sem);
//释放与semaphore相关的资源。
int sem_getvalue(&test_sem,&value);
//将信号量的值赋给value
