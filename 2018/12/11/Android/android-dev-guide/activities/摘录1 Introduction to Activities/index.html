摘录1 Introduction to Activities

Unlike programming paradigms in which apps are launched with a main() method, the Android system initiates code in an Activity instance by invoking specific callback methods that correspond to specific stages of its lifecycle.

Android应用的启动并不像一些程序一样有主函数/入口，Android系统中一个activity实例的初始化是根据activity生命周期的阶段确定回调特定的方法来实现的。

In this way, the activity serves as the entry point for an app’s interaction with the user.

因为android应用不像桌面应用，我们可能是因为不同目的启动它，比如直接启动mail和在qq中通过mail转发；所以每个activity都可以作为app的程序入口，所以没有main（）。

You can use the manifest’s tag to control which apps can start a particular activity. A parent activity cannot launch a child activity unless both activities have the same permissions in their manifest. If you declare a element for a particular activity, the calling activity must have a matching element.

可以为你的app的某个activity附上你自定义的permission，这样可以使别人的app在调用你的activity必须做出声明，因为两个activity必须拥有相同的permission才可以调用。
